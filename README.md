# CanchaLibre
CanchaLibre es un sistema web que permite agilizar el proceso de reserva de canchas deportivas. Contará con una arquitectura que incluye una base de datos para el almacenamiento de usuarios, reservas, canchas e implementos deportivos, un desarrollo backend encargado de la lógica de negocio y un frontend que permitirá la interacción con los usuarios de forma intuitiva mostrando información de canchas y disponibilidad en tiempo real. Estas tecnologías permitirán que los usuarios puedan registrarse e iniciar sesión de manera segura, explorar canchas disponibles filtrando por tipo de deporte, ubicación o precio, y visualizar calendarios de disponibilidad para realizar reservas en horarios específicos. Además, los usuarios podrán gestionar sus reservas, cancelarlas hasta dos horas antes de la hora reservada, recibir notificaciones sobre cambios o confirmaciones y realizar pagos mediante plataformas digitales como Yape y Plin. 
Por su parte, el administrador puede configurar horarios de atención, definir esquemas de precios dinámicos por horario y temporada, y gestionar la disponibilidad de las canchas. Asimismo, incorpora un canal de comunicación directa mediante chat entre usuarios y el administrador, y un sistema de reputación que permite evaluar tanto a los espacios deportivos como a los usuarios, lo que permite a los usuarios tomar decisiones informadas y al administrador gestionar el negocio eficientemente. De esta manera el sistema logrará ofrecer una experiencia de reserva rápida a los clientes y reducir el trabajo del administrador.
